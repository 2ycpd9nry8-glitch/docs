## Overview

The NAS server is a custom built Unraid system designed for high-capacity storage and performance. It runs Unraid version 7.2.1 and provides 84TB of HDD storage with 10.8TB of NVME cache.

## Hardware specifications

### Processor
- **CPU**: Intel Core Ultra 5 265K

### Memory
- **RAM**: 96GB DDR5

### Storage
- **HDD storage**: 84TB
- **NVME cache**: 10.8TB

### Graphics
- **GPU**: Intel Battlemage B580 12GB

### Case
- **Chassis**: Corsair 7000D

## Software

- **Operating system**: Unraid 7.2.1
